== Description ==

Remove / Hide the title of a post from the post edit screen, 
the plugin works with any template and any post type (page, post or custom post type).


== Installation ==

1. Upload the `hide-title` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
